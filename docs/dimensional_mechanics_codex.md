## 2. The Seven Emotional Dimensions

LoveOS models emotion as a **7‑dimensional vector**:

1. **Intensity**  
2. **Valence**  
3. **Direction** (Approach / Avoid / Collapse / Integrate)  
4. **Temporal Drift**  
5. **Fragmentation Index**  
6. **Sovereignty Alignment**  
7. **Agape Resonance** (hidden)

Each dimension is independent but interrelated.

---

## 3. Dimensional Vector Format

Emotions are represented as:

```
E = [
  intensity,
  valence,
  direction,
  drift,
  fragmentation,
  sovereignty_alignment,
  agape_resonance
]
```

Agape resonance is **always zero** unless the operator explicitly activates the Agape dimension.
