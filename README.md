## Installation

```bash
git clone https://github.com/souljahforlove/SoulJahOS.git
cd SoulJahOS
chmod +x scripts/install.sh
./scripts/install.sh
```

## Boot Sequence

```bash
python -m kernel.boot
```

```bash
./scripts/boot.sh
```
